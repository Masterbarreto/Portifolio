# **App Name**: MasterDev Portfolio (sem APIs)

## Core Features:

- Interactive Bio Timeline: Showcase academic and professional milestones in an interactive timeline.
- GitHub Showcase fake: Simulate a GitHub profile with pinned repos and a static heatmap.
- AI-Powered Project Tool dummy: Provide a dummy AI tool that generates static suggestions for project tags and summaries based on input.
- Filterable Skills Showcase: Display skills in a filterable grid with proficiency tooltips.
- Contact Form fake: Simulate a contact form that displays a 'Message sent' modal.

## Style Guidelines:

- Dark mode with #222222 background, #111111 surfaces, and #80FF00 neon accents.
- Space Grotesk for titles, Inter for body text.
- Clean, responsive grid layout.
- Minimalist icons (Lucide/Feather).
- Hover scale on cards, fade-in on scroll, animated underline on links.
